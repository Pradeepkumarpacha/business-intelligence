## 📊 Overview

An advanced Business Intelligence platform that deploys **3 specialized AI agents** to provide comprehensive competitive analysis, market sentiment tracking, and launch performance metrics through a modern Streamlit interface.

### 🎯 Perfect For:
- **GTM Teams** - Competitive positioning and launch strategies
- **Product Marketers** - Market sentiment and competitor analysis  
- **Business Analysts** - Performance metrics and industry insights
- **Executives** - Strategic decision-making with AI-powered reports

## ⚡ Quick Start

### 1. Clone Repository
\\\ash
git clone https://github.com/Pradeepkumarpacha/business-intelligence.git
cd business-intelligence
\\\

### 2. Install Dependencies
\\\ash
pip install -r requirements.txt
\\\

### 3. Configure Environment
Create a \.env\ file:
\\\env
OPENAI_API_KEY=your_openai_api_key_here
FIRECRAWL_API_KEY=your_firecrawl_api_key_here
\\\

### 4. Run Application
\\\ash
streamlit run product_launch_intelligence_agent.py
\\\

### 5. Access Locally
Open: \http://localhost:8501\

## 🎯 How to Use

### Step 1: API Configuration
- Enter your OpenAI API key in the sidebar
- Add your Firecrawl API key for web research

### Step 2: Company Analysis
- Input any company name (e.g., "Tesla", "OpenAI", "Notion")
- Select from three analysis types:

### Step 3: Choose Analysis Type
| Tab | What You Get | Use Case |
|-----|--------------|----------|
| **🔍 Competitor Analysis** | Strategic positioning, launch tactics, strengths/weaknesses | Competitive strategy planning |
| **💬 Market Sentiment** | Social media buzz, customer feedback, sentiment drivers | Brand perception monitoring |
| **📈 Launch Metrics** | Adoption rates, press coverage, performance KPIs | Launch success measurement |

### Step 4: Generate Reports
- Click analysis buttons to generate AI-powered insights
- Receive professional, executive-ready reports with sources
